Project Showcase Challenge

Projects

Drone CV

How can you use a drone to track objects or people? The team of the Drone Tracking study group have researched use cases, implemented tutorials and tried to find information how to integrate federated learning and differential privacy with a drone.

Lane Detection

This project focuses on identification of lanes of the road. Using various approaches and frameworks including but not limited to OpenCV and PyTorch. This model is to be used in drones, self driving cars and surveillance systems.

Crowd density

The main objective of this project is to build a self-sufficient software, that can accurately predict whether or not a stampede is imminent, based on the incoming live visuals of a place.

Face detection

The objective of the project is to calculate the similarity between two facial images. We forseee this application being used for recognizing known faces like in a visual attendence system or as a face based login.

Finder Drone

A project to find anything with a drone! Use case is with cats, drone will find a cat based on videos of it. Please check the summarized presentation video.

ABOUT US

These scholars have been the organisers of the group: Mahak Bansal, Shashi Gharti, Joyce Obi, Seeratpal K. Jaura and Helena Barmer. The group eventually divided into subgroups working on implementing real time object detection and tracking.
